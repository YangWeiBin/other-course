# Web开发（二）

>（2018~2019 第一学期）

《Web开发（二）》是计算机软件及相关专业在Web开发技术方面的基础课，它是为培养应用型人才掌握使用计算机的技能而开设的。本课程是一门有关网页客户端脚本语言的基础课程。本课程的任务是使学生具有使用JavaScript语言和主流的IDE工具编写常用网页程序和实现常用网页特效的能力，并为之后深入学习Web开发打下基础。通过本课程的学习，学生可以熟练掌握JavaScript脚本语言的使用和相关工具的使用。同时，兼顾实用软件的使用和计算机应用领域前沿知识的介绍，为学生进一步学习计算机有关知识打下坚实基础。
